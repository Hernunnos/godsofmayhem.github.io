Source code for iOS games on github

  • 2048 - A SpriteKit based version of the game 2048 source code
  • Castle Hassle - iTunes - source code
  • Aeropack - A steampunk themed retro-platform game created using GameSalad. iTunes - source code
  • Alphabet Blocks - An educational toy featuring letter blocks created with using Cocos2D. iTunes - source code
  • Canabalt - Popular run and jump type game. iTunes - source code
  • Clearis - Cocos2D based puzzle game. iTunes - source code
  • Climbers - 2D finger dragging climbing game created with Cocos2D. iTunes - source code
  • Cryptose - A cryptogram puzzle game made with the Marmalade (formerly Airplay) SDK. iTunes - source code
  • Gorillas – Classic Worms/iShoot turn based shooter type game converted to iPhone from basic. Utilizes Cocos2D. iTunes - source code
  • iLabyrinth - Puzzle game written in Cocos2D requiring you to navigate through increasingly difficult labyrinths. iTunes - source code
  • Little Go - An iOS version of the game of Go featuring a tough AI opponent. iTunes - source code
  • Pteradactyl Attack - An iOS version of the game of Go featuring a tough AI opponent. iTunes - source code
  • Seven Bridges – A puzzle game requiring the player to cross bridges within a specified number of turns. iTunes - source code
